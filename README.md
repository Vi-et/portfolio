# Backend Portfolio (Nuxt 4 + GSAP)

A high-performance, brutalist-style portfolio designed for **Backend Engineers**. This project focuses on technical excellence, minimalist aesthetics, and brutalist typographic design.

> [!NOTE]
> This project serves as a practical space for exploring and mastering **AI Agents** and **Vibe Coding** workflows. Most of the architecture and implementation were co-piloted using advanced agentic coding patterns.

## 🤖 Agentic Engineering & Vibe Coding

This portfolio is not just a static site; it is a demonstration of **Agentic Workflows**. Here is how the AI Agent (Antigravity) was utilized:

*   **Socratic Gate Protocol**: Every major architectural decision (project cleaning, market focus alignment) was preceded by deep discovery questions to ensure technical accuracy and business impact.
*   **Vibe Coding Philosophy**: Beyond just logic, we prioritized "vibe-centric" development—focusing on premium typography, GSAP-powered motion design, and a curated dark-obsidian aesthetic.
*   **Modular Skill Architecture**: Leveraged specialized internal "skills" for:
    *   **`frontend-specialist`**: For brutalist UI/UX and dark mode excellence.
    *   **`gsap-expert`**: For high-performance, non-linear scrolling animations.
    *   **`i18n-localization`**: For seamless, build-safe multi-language synchronization.
*   **No Placeholders Policy**: Every detail reflects real enterprise experience (CRM, Meta Ads API, Advanced Django ORM) rather than boilerplate text.

## 🚀 Key Features

*   **Nuxt 4 Architecture**: Utilizing the latest Nuxt features for high performance and modularity.
*   **GSAP Animations**: Fluid, high-end motion design and scroll-triggered technical visualizations.
*   **Multi-language Support**: Fully localized in **English** and **Vietnamese** using `@nuxtjs/i18n`.
*   **Brutalist Design**: A unique "vibe-centric" UI focusing on typography, sharp edges, and a dark obsidian aesthetic.
*   **Responsive & Optimized**: Built with mobile-first principles and Core Web Vitals in mind.

## 🛠️ Tech Stack

*   **Frontend**: Nuxt 4, Vue 3, Tailwind CSS
*   **Motion**: GSAP (GreenSock Animation Platform)
*   **Localization**: Nuxt i18n
*   **Deployment**: Ideal for Vercel, Netlify, or custom Docker environments.

## 📦 Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## 🛠️ Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## 🏗️ Production

Build the application for production:

```bash
npm run build
```

Locally preview the production build:

```bash
npm run preview
```
